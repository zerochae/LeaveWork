# LeaveWork

> 터미널에 퇴근까지 남은 시간 출력

## Install

1. 프로젝트 클론
```
git clone this repo
```

2. 권한 부여

```
chmod 777 date.sh
```

3. alias 등록

- 클론한 repo에서 `pwd` 입력
  ```sh
  vim ~/.zshrc
  ```

  ```vim
  alias lw {repo path} --> pwd 복붙하기 
  ```

  ```sh
  lw 
  ```
